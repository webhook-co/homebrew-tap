# webhook-co/homebrew-tap

Homebrew tap for **`wbhk`** — the [webhook.co](https://webhook.co) CLI.

```sh
brew install webhook-co/tap/wbhk
```

`wbhk` captures, inspects, and replays webhooks from your terminal. The formula installs a prebuilt,
checksum-pinned binary from the [GitHub releases](https://github.com/webhook-co/webhook/releases).

`Formula/wbhk.rb` is **generated** (and bumped on each release) by `packages/cli/scripts/gen-homebrew-formula.mjs`
in [webhook-co/webhook](https://github.com/webhook-co/webhook) — don't edit it by hand.

Apache-2.0.
